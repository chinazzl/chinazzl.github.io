SpringBoot的启动方式

Jul 11, 2017 | 技术 | 阅读

SpringBoot三种启动方式

  1. Spring的内嵌Tomcat容器 – 最常见、推荐的使用方式

    在你项目搭建完后需要创建一个启动器，通过这个启动器使用SpringBoot内嵌的Tomcat进行启动服务

  2. 第二种使用的是Maven命令进行启动

    启动cmd进入工作空间使用命令行mvn spring-boot:run 进行启动

  3. 第三种使用打包的方式

    • 先将程序进行安装，进入工作空间，使用命令行mvn install
    • 在进入target文件夹 cd target 你会发现已经打好的jar包
    • 使用java命令java -jar xxx.jar 进行启动

以上就是Spring Boot常用的几种启动方式
